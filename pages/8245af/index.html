# 1. жҰӮеҝө

1 дёӘжЁЎеқ—еҸҜиғҪжңүеӨҡдёӘвҪ…жі•пјҢеҸӘиҰҒе…¶дёӯзҡ„жҹҗдёӘвҪ…жі•дҪҝвҪӨеҲ°дәҶпјҢеҲҷж•ҙдёӘвҪӮд»¶йғҪдјҡиў«жү“еҲ° bundle вҫҘвҫҜеҺ»пјҢtree shaking е°ұжҳҜеҸӘжҠҠвҪӨеҲ°зҡ„вҪ…жі•жү“вјҠ bundle пјҢжІЎвҪӨеҲ°зҡ„вҪ…жі•дјҡеңЁ uglify йҳ¶ж®өиў«ж“ҰйҷӨжҺүгҖӮ

  • return еҗҺйқўзҡ„д»Јз Ғ
  • еҸӘеЈ°жҳҺпјҢжңӘдҪҝз”Ёзҡ„еҮҪж•°
  • еҸӘеј•е…ҘпјҢжңӘдҪҝз”Ёзҡ„д»Јз Ғ

# 2. йҷҗеҲ¶

  • дҪҝз”ЁES Moduleи§„иҢғжЁЎеқ—пјҢпјҢжүҚиғҪжү§иЎҢTree Shaking
  • Tree Shakingдҫқиө–дәҺES Modulesзҡ„йқҷжҖҒиҜӯжі•еҲҶжһҗ

# 3. Tree-shaking еҺҹзҗҶ

  • еҲ©вҪӨ ES6 жЁЎеқ—зҡ„зү№зӮ№:

    • еҸӘиғҪдҪңдёәжЁЎеқ—йЎ¶еұӮзҡ„иҜӯеҸҘеҮәзҺ°
    • import зҡ„жЁЎеқ—еҗҚеҸӘиғҪжҳҜеӯ—з¬ҰдёІеёёйҮҸ
    • import binding жҳҜ immutableзҡ„ д»Јз Ғж“ҰйҷӨпјҡ uglify йҳ¶ж®өеҲ йҷӨвҪҶвҪӨд»Јз Ғ

  • ејҖеҸ‘жЁЎејҸпјҡ

    • usedExports
    • sideEffects

зҺ°иұЎпјҡжһ„е»әеҗҺзҡ„д»Јз ҒеӯҳеңЁвјӨйҮҸй—ӯеҢ…д»Јз Ғ

еҜјиҮҙ:

  • вјӨйҮҸдҪңвҪӨеҹҹеҢ…иЈ№д»Јз ҒпјҢеҜјиҮҙдҪ“з§ҜеўһвјӨпјҲжЁЎеқ—и¶ҠеӨҡи¶ҠжҳҺжҳҫпјү

  • иҝҗвҫҸд»Јз Ғж—¶еҲӣе»әзҡ„еҮҪж•°дҪңвҪӨеҹҹеҸҳеӨҡпјҢеҶ…еӯҳејҖй”ҖеҸҳвјӨ

  • жЁЎеқ—иҪ¬жҚўеҲҶжһҗ:

  • иў« webpack иҪ¬жҚўеҗҺзҡ„жЁЎеқ—дјҡеёҰдёҠвјҖеұӮеҢ…иЈ№

  • import дјҡиў«иҪ¬жҚўжҲҗ __webpack_require

  • жү“еҢ…еҮәжқҘзҡ„жҳҜвјҖдёӘIIFE (еҢҝеҗҚй—ӯеҢ…)

  • modules жҳҜвјҖдёӘж•°з»„пјҢжҜҸвјҖйЎ№жҳҜвјҖдёӘжЁЎеқ—еҲқе§ӢеҢ–еҮҪж•°

  • йҖҡиҝҮ WEBPACK_REQUIRE_METHOD(0) еҗҜеҠЁзЁӢеәҸ

  • __webpack_require вҪӨжқҘеҠ иҪҪжЁЎеқ—пјҢиҝ”еӣһ module.exports

# 4. scope hoisting еҺҹзҗҶ

  • еҺҹзҗҶпјҡе°ҶжүҖжңүжЁЎеқ—зҡ„д»Јз ҒжҢүз…§еј•вҪӨйЎәеәҸж”ҫеңЁвјҖдёӘеҮҪж•°дҪңвҪӨеҹҹвҫҘпјҢз„¶еҗҺйҖӮеҪ“зҡ„йҮҚе‘ҪеҗҚвјҖ дәӣеҸҳйҮҸд»ҘйҳІвҪҢеҸҳйҮҸеҗҚеҶІзӘҒ

  • еҜ№вҪҗ: йҖҡиҝҮ scope hoisting (дҪңз”ЁеҹҹжҸҗеҚҮ)еҸҜд»ҘеҮҸе°‘еҮҪж•°еЈ°жҳҺд»Јз Ғе’ҢеҶ…еӯҳејҖй”Җ

  • scope hoisting дҪҝвҪӨпјҡ webpack mode дёә production й»ҳи®ӨејҖеҗҜ,еҝ…йЎ»жҳҜ ES6 иҜӯжі•пјҢCJS дёҚвҪҖжҢҒ

# sideEffects

  • еүҜдҪңз”Ё:
    • ж— еүҜдҪңз”ЁпјҡеҰӮжһңдёҖдёӘжЁЎеқ—еҚ•зәҜзҡ„еҜје…ҘеҜјеҮәеҸҳйҮҸпјҢйӮЈе®ғе°ұж— еүҜдҪңз”Ё
    • жңүеүҜдҪңз”ЁпјҡеҰӮжһңдёҖдёӘжЁЎеқ—иҝҳдҝ®ж”№е…¶д»–жЁЎеқ—жҲ–иҖ…е…ЁеұҖзҡ„дёҖдәӣдёңиҘҝпјҢе°ұжңүеүҜдҪңз”Ё
      • дҝ®ж”№е…ЁеұҖеҸҳйҮҸ
      • еңЁеҺҹеһӢдёҠжү©еұ•ж–№жі•
      • cssзҡ„еј•е…Ҙ
  • sideEffectsзҡ„дҪңз”ЁпјҡжҠҠжңӘдҪҝз”ЁдҪҶж— еүҜдҪңз”Ёзҡ„жЁЎеқ—дёҖе№¶еҲ йҷӨ
    • еҜ№дәҺжІЎжңүеүҜдҪңз”Ёзҡ„жЁЎеқ—пјҢжңӘдҪҝз”Ёд»Јз ҒдёҚдјҡиў«жү“еҢ…пјҲзӣёеҪ“дәҺеҺӢзј©дәҶиҫ“еҮәеҶ…е®№пјү
